Game starts in starting screen, press Start (Enter) to begin game



Controls:
Up arrow    ->	move up on screen
Down arrow	->	move down on screen
Left arrow	->	move left on screen
Right arrow ->	move right on screen

SELECt (backspace) 	-> return to starting screen



Directions:

Move frog to opposite side of screen in 30 seconds.
Avoid falling into the water by stepping on the moving logs.
Counter on bottom left shows time elapsed
